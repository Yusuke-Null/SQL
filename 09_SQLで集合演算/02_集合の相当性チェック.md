# 集合の相当性チェック
## テーブル同士のコンペア- 集合の相当性チェック基本編
DB環境を移行したり、バックアップと最新環境を比較するなど2つのテーブルが等しいかチェックしたい場合がある  
-> 等しいとは行数も列数もデータの内容も同じ = 集合として等しい
