| 販売実績テーブル       |        |         |   | 商品マスタ |           |

| sale_date              | item_no| quantity|   | item_no    | item      |
| 2018-10-01             | 10     | 4       |   | 10         | SDカード   |
| 2018-10-01             | 20     | 10      |   | 20         | CD-R      |
| 2018-10-01             | 30     | 3       |   | 30         | USBメモリ |
| 2018-10-03             | 10     | 32      |   | 40         | DVD       |
| 2018-10-03             | 30     | 12      |   |            |           |
| 2018-10-04             | 20     | 22      |   |            |           |
| 2018-10-04             | 30     | 7       |   |            |           |
